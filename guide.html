3. 필터 사용법3. Using Filters

필터는 캐스케이딩(계단식) 방식으로 작동합니다 — 각 선택에 따라 다음 필터의 옵션이 자동으로 좁혀집니다.

Filters work in a cascading manner — each selection narrows down the options in the next filter.

4. 조회 및 비교4. Search & Compare

운임을 조회하는 두 가지 방법이 있습니다:

There are two ways to view rates:

Search

Active routes appear as colored tags below the filter bar. Click x to remove a route, or Clear All to reset.

5. 운임 요약 카드5. Rate Summary Cards

각 활성 루트에 대해 최신 운임, 전주 대비 변동, 이력 범위, 그리고 원본 파일의 운임 조건(Remark)을 요약 카드로 표시합니다.

For each active route, a summary card shows the latest rate, week-over-week change, historical range, and rate conditions (remarks) from the source file.

카드 구성 요소Card Elements

  • 루트 라벨 — POL -> POD (컨테이너 크기)
  • 최신 운임 — 가장 최근 주차의 USD 운임
  • 변동 — ▲ 빨간색 = 상승, ▼ 초록색 = 하락
  • 범위 — 전체 기간의 최소 ~ 최대 운임
  • Remark (파란 박스) — 원본 Excel 파일의 운임 조건 (할증료, 포함 항목 등)
  • Route label — POL -> POD (container size)
  • Latest rate — Most recent week's rate in USD
  • Change — ▲ Red = increase, ▼ Green = decrease
  • Range — Min ~ Max rate across all available weeks
  • Remark (blue box) — Rate conditions from the source Excel file (surcharges, inclusions, etc.)

다수 루트 요약 모드Multi-Route Summary Mode

7개 이상의 루트를 비교할 때는 요약 카드로 자동 전환됩니다: 총 루트 수(주차 표시), 평균 운임, 최고/최저. 모든 운임은 현재 주차 기준으로 표시됩니다.

When comparing 7+ routes, cards switch to summary mode: total routes (with week label), average rate, highest/lowest. All rates reflect the current week.

참고:Note: 도착지 고정 비교(같은 POD, 다른 POL) 시 최고/최저에는 선적지(POL) 포트명이 표시됩니다. 선적지 고정 비교(같은 POL, 다른 POD) 시에는 도착지(POD) 포트명이 표시됩니다. When comparing to the same destination (fixed POD, different POLs), highest/lowest shows the origin port name. When comparing from the same origin (fixed POL, different PODs), it shows the destination port name.

6. 운임 추이 차트6. Rate Trend Chart

인터랙티브 라인 차트로 시간에 따른 운임 추이를 보여줍니다. 각 루트는 태그 색상과 일치하는 다른 색상으로 표시됩니다.

An interactive line chart shows rate trends over time. Each route is a different color matching its tag.

주요 기능Features

  • 기간 선택 버튼 — All, 1Y (52주), 6M (26주), 3M (13주)
  • 마우스 호버 툴팁 — 해당 주차와 모든 루트 운임을 높은 순서대로 정렬하여 표시합니다. 해당 주차에 데이터가 없는 루트는 숨겨집니다.
  • X축 라벨 — 형식: W15 '26 (주차 번호 + 연도)
  • Time range buttons — All, 1Y (52 weeks), 6M (26 weeks), 3M (13 weeks)
  • Hover tooltip — Shows the week and all route rates, sorted from highest to lowest. Routes with no data for that week are hidden.
  • X-axis labels — Format: W15 '26 (week number + year)

8. All 도착지 비교 기능8. All Destination Comparison

POD Country와 POD Port 필터에서 All을 선택하면 선택한 선적지(POL)의 모든 도착지를 한번에 비교할 수 있습니다.

Select All in the POD Country and POD Port filters to compare all destinations from the selected origin(s) at once.

자동 확장 동작Auto-Expand Behavior

필터 상태동작 Filter StateBehavior
POD Country=All, Port=All모든 도착 국가별로 자동 확장 POD Country=All, Port=AllAuto-expands by destination country
POD Country=KR, Port=All해당 국가 내 모든 포트별로 자동 확장 POD Country=KR, Port=AllAuto-expands by all ports within that country
POD Country=KR, Port=PUS단일 루트 조회 (기존 동작) POD Country=KR, Port=PUSSingle route lookup (original behavior)

Destination Pivot TableDestination Pivot Table

같은 선적지에서 5개 이상 도착지를 비교하면, 테이블이 자동으로 Destination Pivot 형식으로 전환됩니다:

When comparing 5+ destinations from the same origin, the table automatically switches to Destination Pivot format:

  • 행 = 도착지 (최신 운임 높은 순 정렬)
  • 열 = 최근 10주 + WoW 변동 + 운임 범위
  • 모든 열 헤더 클릭으로 정렬 가능
  • Rows = Destinations (sorted by latest rate, highest first)
  • Columns = Latest 10 weeks + WoW change + Rate range
  • Click any column header to sort

지원 선적지 (POL)Supported Origins (POL)

SEA 소스에서 19개 선적지를 지원합니다. TH (방콕/렘차방), PH (마닐라/MIP), MY PEN (페낭) 포함.

19 origin ports are supported from SEA source, including TH (Bangkok/Laem Chabang), PH (Manila/MIP), and MY PEN (Penang).
